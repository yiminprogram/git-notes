## Setting email & name

- global setting

```zsh
git config --global user.name "user name"

git config --global user.email "user email"
```

- local setting

```zsh
git config --local user.name "user name"

git config --local user.email "user email"
```

## Command Alias(別名)

```zsh
git config --global alias.st status

# git status -> git st
```
